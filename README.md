# Wells Chaos Calendar

A family trip planning app for Disney World and Universal Orlando adventures.

## Getting started

### Install dependencies

```bash
npm install
```

### Run the app

```bash
npm run dev
```

### Build for production

```bash
npm run build
```

### Run tests

```bash
npm run test
```

## Project structure

- `src/components/WellsChaosCalendar/` contains the main UI and subcomponents.
- `src/data/` contains theme park data and presets.
- `src/types/` contains shared TypeScript types.
- `src/App.tsx` wires the component into the app shell.
- `src/index.css` contains Tailwind CSS setup.
